Arquitectura de la simulacion:

├── main.py              ← Archivo principal (loop de glfw)
├── config.py            ← Constantes y parámetros
├── camera.py            ← Cámara y controles
├── physics.py           ← Física, Luna, partículas
├── rendering.py         ← Todo lo relacionado con ModernGL (shaders, buffers, render)
├── entities.py          ← Clases para Tierra, Luna, Partícula
└── utils.py             ← Funciones auxiliares (texto, format, etc.)
